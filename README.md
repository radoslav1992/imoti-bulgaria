# Имоти България — Средни цени и статистики

Уеб приложение за проследяване на цените на недвижими имоти в България, базирано на официалните средни цени от [imot.bg](https://www.imot.bg/sredni-ceni). Данните се извличат автоматично всеки ден чрез GitHub Actions.

## Функционалности

- **16 града** — средни цени на м² подредени по стойност
- **480+ квартала** — детайлни цени по квартали за всеки град
- **По тип имот** — разбивка за 1-стаен, 2-стаен, 3-стаен и общо EUR/м²
- **Тенденции** — графика с история на цените (Chart.js)
- **Интерактивност** — кликнете на град за да видите кварталите му
- **Сгъваеми секции** — всички секции могат да се сгъват/разгъват
- **Любими** — запазете градове и квартали в localStorage, видими на отделна страница `/favorites`
- **PWA** — инсталируемо приложение с офлайн поддръжка

## Покрити градове

София, Варна, Бургас, Пловдив, Шумен, Велико Търново, Перник, Стара Загора, Плевен, Ямбол, Русе, Благоевград, Сливен, Хасково, Пазарджик, Добрич

## Технологии

- [Astro](https://astro.build/) — статичен сайт генератор
- [Chart.js](https://www.chartjs.org/) — графики
- [Cheerio](https://cheerio.js.org/) — HTML парсване
- [iconv-lite](https://github.com/ashtuchkin/iconv-lite) — windows-1251 декодиране
- GitHub Actions — автоматизация

## Започване

```bash
npm install
npm run dev        # localhost:4321
npm run scrape     # извличане на данни от imot.bg
npm run build      # билд за продукция
```

## Как работи скрейпърът

Скрейпърът извлича данни от `imot.bg/sredni-ceni/prodazhbi-{град}` — страницата с официалните средни цени по квартали. За всеки град се парсва HTML таблицата с:

- Име на квартала
- Цена и EUR/м² за 1-стаен, 2-стаен и 3-стаен апартаменти
- Обща средна цена на EUR/м²

## GitHub Actions

Изпълнява се ежедневно в 06:00 UTC:

1. Извлича средни цени от imot.bg за всеки град
2. Записва данните и генерира история
3. Билдва и деплойва на GitHub Pages

## Конфигурация

Градовете се настройват в `scripts/scrape.mjs` → масива `CITIES`.
